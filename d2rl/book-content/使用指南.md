## 💻 代码实践环境配置

要运行 Hands-on-RL/ 目录中的 Jupyter Notebook 代码，需要配置 Python 环境：

### 1. 安装依赖

```bash
cd /Users/song/Desktop/code/RL\ learning/d2rl/Hands-on-RL

# 创建虚拟环境（推荐）
python3 -m venv rl_env
source rl_env/bin/activate

# 安装必要的库
pip install numpy matplotlib torch gymnasium jupyter
```

### 2. 启动 Jupyter Notebook

```bash
jupyter notebook
```

然后在浏览器中选择对应的章节进行学习。

## 📝 学习建议

### 理论 + 实践结合

1. **先读理论**：在 `book-content/` 中阅读对应章节的 HTML 或 Markdown 文件
2. **再跑代码**：在 `Hands-on-RL/` 中找到对应的 Jupyter Notebook 运行实验
3. **动手修改**：尝试修改代码参数，观察不同效果
